These series of instructions will walk you though the installation of core packages needed for running the validation framework, beginning with the installing sciunit.

Purpose


Install sciunit.
The validation framework is based on sciunit.
The sciunit package is available from https://github.com/scidash/sciunit
 

How to.


To install sciunit in the notebook,
>> !pip install https://github.com/scidash/sciunit/archive/master.zip

 

Purpose


Install hbp-validation framework.
This is available from https://github.com/apdavison/hbp-validation-client.
 

How to.


To install hbp-validation framework in the notebook,
>> !pip install https://github.com/apdavison/hbp-validation-client/archive/master.zip

 

To run the VUCs (Validation Use Cases) in this collaboratory page there are additional packages needed (they may already be installed in your own collab. page). This page shows the installation of eFEL (Electrophys Feature Extraction Library).

Purpose


Install eFEL.
This is available from https://github.com/BlueBrain/eFEL.
 

How to.


To install eFEL in the notebook,
>> !pip install https://github.com/BlueBrain/eFEL/archive/master.zip

 

If its not already installed you will also need to have git which gets imported in some validation test scripts and also used in this walkthrough.

Purpose


Install git.
Link to an external website. The website is opened within the iframe if whitelisted. Otherwise, it opens in a separate window. This is available from https://github.com/gitpython-developers/GitPython or from https://pypi.python.org/pypi/GitPython/
 

How to.


To install git in the notebook,
>> !pip install https://github.com/gitpython-developers/GitPython/archive/master.zip
