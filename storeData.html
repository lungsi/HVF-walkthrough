Storing the Data of the Validation Test Result.


Purpose

Store the data of the results from the running the desired validation test on the chosen model.

 

How to.

To store the results data of our test run on the collab storage do the following,

>> from hbp_validation_framework.datastores import CollabDataStore
>> collab_storage = CollabDataStore(username="lungsi", collab_id="1655", base_folder="VUC_Bianchi_DepolTest_results")

 

The above command stores the results files in the directory named VUC_Bianchi_DepolTest_results of BSP collab (1655 id) storage. This is shown in the figure below.
