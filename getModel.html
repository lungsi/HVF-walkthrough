Choosing and Loading a Model.

Purpose

From the HVF list of models the user chooses a model he/she is interested in running the desired validation test.

How to.

Let's say we are interested in the hippocampus models. In particular, a model called Bianchi-CA1 pyramidal cell model listed in the HVF model catalog. From the details under this model we see that the code of the model is located in https://github.com/apdavison/hippocampus_CA1_pyramidal. Therefore,

 

Purpose

Get hippocampal model collection containing the chosen Bianchi-CA1 pyramidal cell model. That is clone hippocomapus_CA_pyramidal.

How to.

The hippocampus_CA1_pyramidal is available from https://github.com/apdavison/hippocampus_CA1_pyramidal
 
Assuming you are in the desired working directory type the command below
>> !git clone https://github.com/apdavison/hippocampal_CA1_pyramidal

Purpose

Check that hippocomapus_CA_pyramidal has been cloned and then enter the directory.

How to.

See figure on the right to check that the directory hippocomapus_CA_pyramidal is present.
Enter the directory hippocomapus_CA_pyramidal.
Notice the presence of a model module.

Purpose

Import the models module and extract the list of models available for running validation tests.

How to.

Import the models module. Notice that to do this the user should be in the directory path_to/hippocampus_CA1_pyramidal
To get the list of available models do
>> import inspect
>> [m[0] for m in inspect.getmembers(models, inspect.isclass) if m[1].__module__ == 'models']
In this example the models are:
  • Bianchi
  • Golding
  • KaliFreund
  • Migliore
