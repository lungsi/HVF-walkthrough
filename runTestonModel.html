Running the Desired Validation Test On the Chosen Model.

At this step the user

  • has considered the model chosen to validate.
    For our example: Bianchi (hippocampus CA1 pyramidal cell model)
  • has also downloaded the desired v-test; this has the defined v-test and the data against which the test will be compared.

Purpose

Run the desired validation test on the chosen model.

 

How to.

The figure illustrates the setup need prior to running the v-test.
Once the setup is done, do
>> score = test.judge(getattr(models, 'Bianchi')(), deep_error=True)
